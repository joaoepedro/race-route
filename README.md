# AutoManage - Sistema de Gestão para Autocenters

## 📋 Descrição

AutoManage é um sistema integrado para gerenciamento de estoque e agendamentos de serviços automotivos. Desenvolvido para otimizar operações de centros automotivos, esta plataforma simplifica o controle de inventário, agendamentos de serviços e atendimento ao cliente em uma interface unificada.

## 🖥️ Funcionalidades Principais

- **Controle de Estoque**:
  - Cadastro e atualização de produtos e peças
  - Alertas de estoque baixo
  - Histórico de entradas e saídas
  - Integração com fornecedores

- **Agendamentos**:
  - Calendário de serviços
  - Registro de clientes
  - Histórico de veículos
  - Notificações automáticas

- **Gestão Financeira**:
  - Orçamentos e notas fiscais
  - Controle de pagamentos
  - Relatórios gerenciais
  - Análise de lucratividade

## 💻 Tecnologias Utilizadas

- **Frontend**: HTML e CSS
- **Backend**: PHP
- **Banco de Dados**: MySQL
